# 📚 Dokumentasi Pembelajaran Dasar React JS

Project ini berisi implementasi dan contoh penggunaan konsep-konsep dasar React JS menggunakan **Vite**.

---

## 📌 Konsep Dasar yang Diimplementasikan

1. **Pengertian React JS & Vite**: Library JavaScript berbasis komponen untuk membangun UI interaktif, di-setup menggunakan Vite untuk performa build yang cepat.
2. **JSX (JavaScript XML)**: Ekstensi sintaks JavaScript yang memungkinkan penulisan struktur HTML langsung di dalam file JS/JSX.
3. **Functional Component**: Komponen UI berbasis fungsi JavaScript (`App.jsx` dan `CardProduk.jsx`).
4. **Props**: Pengiriman data dari komponen induk ke komponen anak (seperti nama, harga, dan kategori ke `CardProduk`).
5. **State (`useState`)**: Penyimpanan data dinamis dalam komponen (digunakan pada fitur counter dan toggle section).
6. **Hooks (`useEffect`)**: Menangani efek samping untuk simulasi pengambilan data produk saat komponen pertama kali dimuat.
7. **Event Handling**: Menangani aksi pengguna (seperti `onClick` pada tombol counter dan toggle).
8. **Conditional Rendering**: Menampilkan atau menyembunyikan elemen UI berdasarkan kondisi state (`showSection`).
9. **List & Key**: Menampilkan array data produk menggunakan `.map()` dengan `key` unik untuk setiap item.
